⚙️ 6.2 Ultrasonic Sensor (HC-SR04)
🇬🇧 English Reference (EN)
Description: Measures distance by sending sound waves. It requires a Trig pin (output) to send sound and an Echo pin (input) to receive it.
Key Logic: Distance = (Time x Speed of Sound) / 2.
🇹🇷 Türkçe Referans (TR)
Açıklama: Ses dalgaları göndererek mesafeyi ölçer. Ses göndermek için Trig pini (çıkış) ve almak için Echo pini (giriş) gerektirir.
Temel Mantık: Mesafe = (Süre x Ses Hızı) / 2.
Code / Kod:
code C++
downloadcontent_copy
expand_less
    const int trigPin = 9;
const int echoPin = 10;

void setup() {
  pinMode(trigPin, OUTPUT);
  pinMode(echoPin, INPUT);
  Serial.begin(9600);
}

void loop() {
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);
  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);

  long duration = pulseIn(echoPin, HIGH);
  int distance = duration * 0.034 / 2; // cm

  Serial.print("Dist: ");
  Serial.println(distance);
  delay(500);
}

🚀 Project 7.1: Reverse Parking Sensor (Geri Park Sensörü)
Components / Bileşenler:
1. HC-SR04 Ultrasonic Sensor
2. Buzzer
3. LED (Red / Kırmızı)
4. LED (Green / Yeşil)
🇬🇧 English Explanation
Logic: The system measures distance.
* If the distance is safe (> 20cm): The Green LED is ON, Buzzer is OFF.
* If the distance is danger (< 20cm): The Red LED turns ON, Green turns OFF, and the Buzzer beeps. The closer the object, the faster the beeping (optional logic).
🇹🇷 Türkçe Açıklama
Mantık: Sistem mesafeyi ölçer.
* Mesafe güvenli (> 20cm) ise: Yeşil LED AÇIK, Buzzer KAPALI.
* Mesafe tehlikeli (< 20cm) ise: Kırmızı LED AÇIK, Yeşil KAPALI ve Buzzer öter. Cisim ne kadar yakınsa, ötme o kadar hızlanır (opsiyonel mantık).
Code / Kod:
code C++
    const int trigPin = 9;
const int echoPin = 10;
const int buzzer = 11;
const int ledRed = 12;
const int ledGreen = 13;

void setup() {
  pinMode(trigPin, OUTPUT);
  pinMode(echoPin, INPUT);
  pinMode(buzzer, OUTPUT);
  pinMode(ledRed, OUTPUT);
  pinMode(ledGreen, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  // 1. Measure Distance / Mesafeyi Ölç
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);
  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);
  long duration = pulseIn(echoPin, HIGH);
  int distance = duration * 0.034 / 2;

  Serial.print("Distance: ");
  Serial.println(distance);

  // 2. Decision Logic / Karar Mantığı
  if (distance < 20) {
    // Danger Zone / Tehlike Bölgesi
    digitalWrite(ledGreen, LOW);
    digitalWrite(ledRed, HIGH);
    
    // Beep sound / Bip sesi
    tone(buzzer, 1000); 
    delay(distance * 10); // Beep faster as you get closer / Yaklaştıkça hızlan
    noTone(buzzer);
    delay(distance * 10);
    
  } else {
    // Safe Zone / Güvenli Bölge
    digitalWrite(ledRed, LOW);
    digitalWrite(ledGreen, HIGH);
    noTone(buzzer);
    delay(100);
  }
}
  

🌡️ Project 7.2: Automatic Climate Control (Otomatik İklim Kontrolü)
Components / Bileşenler:
1. DHT11 Temperature Sensor
2. I2C LCD Display (16x2)
3. DC Motor (Fan) + L298N Driver
🇬🇧 English Explanation
Logic: A thermostat system.
* The Arduino reads the temperature.
* It displays the current temperature on the LCD screen.
* Condition: If the temperature exceeds 28°C, the Fan (DC Motor) turns ON automatically to cool down. If it drops below 28°C, the fan turns OFF.
🇹🇷 Türkçe Açıklama
Mantık: Bir termostat sistemi.
* Arduino sıcaklığı okur.
* Mevcut sıcaklığı LCD ekranda gösterir.
* Koşul: Eğer sıcaklık 28°C'yi geçerse, Fan (DC Motor) soğutmak için otomatik olarak AÇILIR. 28°C'nin altına düşerse fan KAPANIR.
Code / Kod:
code C++
    #include <Wire.h> 
#include <LiquidCrystal_I2C.h>
#include <DHT.h>

// Pins
#define DHTPIN 2
#define DHTTYPE DHT11
int motorPin = 9; // L298N IN1 or Transistor Base

DHT dht(DHTPIN, DHTTYPE);
LiquidCrystal_I2C lcd(0x27, 16, 2);

void setup() {
  dht.begin();
  lcd.init();
  lcd.backlight();
  pinMode(motorPin, OUTPUT);
}

void loop() {
  // 1. Read Data / Veriyi Oku
  float t = dht.readTemperature();
  
  // 2. Display / Görüntüle
  lcd.setCursor(0, 0);
  lcd.print("Temp: ");
  lcd.print(t);
  lcd.print(" C");

  // 3. Logic / Mantık
  lcd.setCursor(0, 1);
  if (t > 28.0) {
    digitalWrite(motorPin, HIGH); // Fan ON / Fan Açık
    lcd.print("Fan: ON ");
  } else {
    digitalWrite(motorPin, LOW);  // Fan OFF / Fan Kapalı
    lcd.print("Fan: OFF");
  }
  
  delay(2000); // Wait 2 seconds / 2 saniye bekle
}
  

🔐 Project 7.3: Smart Door Lock (Akıllı Kapı Kilidi)
Components / Bileşenler:
1. Servo Motor (The Lock)
2. Button (or Membrane Keypad)
3. Buzzer
4. Red & Green LEDs
🇬🇧 English Explanation
Logic: Simulates a secure door.
* The door is initially LOCKED (Servo at 0°, Red LED ON).
* When the button is pressed (simulating a correct password or fingerprint):
1. Servo moves to 90° (Unlock).
2. Green LED turns ON, Red OFF.
3. Buzzer plays a "Success" melody.
4. Wait 5 seconds, then auto-lock again.
🇹🇷 Türkçe Açıklama
Mantık: Güvenli bir kapıyı simüle eder.
* Kapı başlangıçta KİLİTLİDİR (Servo 0°'de, Kırmızı LED AÇIK).
* Butona basıldığında (doğru şifre veya parmak izi simülasyonu):
1. Servo 90°'ye hareket eder (Kilit Aç).
2. Yeşil LED AÇIK, Kırmızı KAPALI.
3. Buzzer "Başarılı" melodisi çalar.
4. 5 saniye bekle, sonra tekrar otomatik kilitle.
Code / Kod:
code C++
    #include <Servo.h>

Servo myLock;
const int buttonPin = 2;
const int buzzer = 8;
const int redLED = 12;
const int greenLED = 13;

void setup() {
  myLock.attach(9);
  pinMode(buttonPin, INPUT_PULLUP); // Button to GND / Buton Toprağa bağlı
  pinMode(buzzer, OUTPUT);
  pinMode(redLED, OUTPUT);
  pinMode(greenLED, OUTPUT);
  
  // Initial State: Locked / Başlangıç Durumu: Kilitli
  myLock.write(0);
  digitalWrite(redLED, HIGH);
  digitalWrite(greenLED, LOW);
}

void loop() {
  // If button pressed (Active LOW) / Butona basılırsa (Aktif DÜŞÜK)
  if (digitalRead(buttonPin) == LOW) {
    unlockDoor();
  }
}

void unlockDoor() {
  // 1. Visual & Audio / Görsel & İşitsel
  digitalWrite(redLED, LOW);
  digitalWrite(greenLED, HIGH);
  tone(buzzer, 1500, 200); // Beep
  
  // 2. Open Lock / Kilidi Aç
  myLock.write(90);
  
  // 3. Wait / Bekle
  delay(5000); // Keep open for 5 seconds / 5 saniye açık tut
  
  // 4. Lock Again / Tekrar Kilitle
  myLock.write(0);
  digitalWrite(greenLED, LOW);
  digitalWrite(redLED, HIGH);
  tone(buzzer, 500, 500); // Lock sound / Kilitlenme sesi
}
